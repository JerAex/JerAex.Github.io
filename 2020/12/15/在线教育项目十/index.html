二、SpringCloud

2.1 微服务

  • 微服务是一种架构风格
  • 有多个服务，多个服务是独立运行，每个服务占用独立进程

2.2 SpringCloud简介

  • SpringCloud并不是一种机制，是很多技术的集合
  • SpringCloud里面有很多框架，使用SpringCloud里面这些框架实现微服务操作
  • 使用SpringCloud，需要依赖技术Springboot

2.3 SpringCloud相关基础服务组件

  • 服务发现：Netflix Eureka（Nacos）
  • 服务调用：Netflix Feign
  • 熔断器：Netflix Hystrix
  • 服务网关：Spring Cloud GateWay
  • 分布式配置：Spring Cloud Config（Nacos）
  • 消息总线：Spring Cloud Bus（Nacos）
